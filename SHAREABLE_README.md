# LocalRAGChatBot v2.0.0

Thank you for downloading LocalRAGChatBot.

## Included

-   LocalRAGChatBot.exe
-   Required DLL files

## Installation

1.  Extract all files into one folder.
2.  Do **not** move the DLL files.
3.  Place your GGUF chat and embedding models on your machine.
4.  Configure model paths if required.
5.  Launch `LocalRAGChatBot.exe`.

## Other Platforms

Linux and macOS users should clone the repository and build from source.

Repository: https://github.com/SatyaHimavanth/LocalRAGChatBot
